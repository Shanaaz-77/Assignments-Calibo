# Assignments 

## Overview
This repository contains a collection of assignments completed as part of the Calibo AI Training Program. It covers fundamental and advanced concepts in Python programming, data handling, machine learning, MLOps, and retrieval-based AI systems.

The repository is organized into multiple folders, each representing a specific module or topic, demonstrating practical implementation and understanding of core concepts.

## Objectives
- Build a strong foundation in Python programming
- Work with data using NumPy and Pandas
- Perform data cleaning and exploratory analysis
- Understand and implement machine learning models
- Learn MLOps concepts such as data versioning and experiment tracking
- Develop knowledge of modern AI systems like RAG pipelines

## Repository Structure


Assignments/
├── Python Lab/ # Python basics, functions, lists, dictionaries
├── Data Handling/ # NumPy, Pandas, data cleaning, EDA
├── Machine Learning/ # Regression, evaluation, confusion matrix
├── MLOPS/ # DVC, MLflow, model versioning, lifecycle
└── RAG/ # Retrieval-Augmented Generation pipeline


## Modules Description

### Python Lab
Covers core Python concepts including syntax, variables, functions, lists, and dictionaries. Builds the programming foundation required for further modules.

### Data Handling
Focuses on data processing using NumPy and Pandas, including data cleaning, transformation, and exploratory data analysis.

### Machine Learning
Includes implementation of basic machine learning models and evaluation techniques such as regression, confusion matrix, and performance metrics.

### MLOPS
Demonstrates a complete machine learning workflow using concepts like data versioning (DVC), experiment tracking (MLflow), model versioning, and lifecycle management.

### RAG
Implements a Retrieval-Augmented Generation pipeline to combine information retrieval with generative AI for context-aware responses.
