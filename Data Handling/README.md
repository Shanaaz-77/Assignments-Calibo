# Data Handling: NumPy, Pandas, and Exploratory Data Analysis

## Overview
This project focuses on essential data handling techniques using Python. It covers numerical computations with NumPy, data cleaning with Pandas, and exploratory data analysis (EDA) to understand and summarize datasets.

## Objective
- Perform numerical operations using NumPy
- Clean and preprocess data using Pandas
- Analyze and visualize data through exploratory techniques
- Build a strong foundation for data science workflows

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib / Seaborn (for EDA, if used)

## Files Included
- Shanaaz_Numpy_Fundamentals.ipynb  
- Shanaaz_Pandas_Data_Cleaning.ipynb  
- Shanaaz_Exploratory_Analysis.ipynb  

## Topics Covered
### NumPy
- Array creation and manipulation
- Indexing, slicing, and reshaping
- Mathematical and statistical operations
- Broadcasting

### Pandas
- Loading datasets
- Handling missing values (`dropna`, `fillna`)
- Removing duplicates
- Data type conversion
- Filtering and sorting

### Exploratory Data Analysis (EDA)
- Data inspection and summary statistics
- Identifying patterns and trends
- Data visualization
- Understanding data distributions

## How It Works
1. Use NumPy for efficient numerical computations
2. Clean and preprocess datasets using Pandas
3. Perform exploratory analysis to extract insights
4. Prepare data for further machine learning tasks

## Installation
Install required libraries:
pip install numpy pandas matplotlib seaborn

## Usage
Run using Jupyter Notebook or Google Colab.

## Learning Outcomes
- Strong understanding of data handling techniques
- Ability to clean and preprocess real-world datasets
- Skills in analyzing and visualizing data
- Foundation for machine learning and data science projects
