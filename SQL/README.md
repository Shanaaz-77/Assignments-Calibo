# SQL 

## Overview
This repository contains a collection of SQL assignments covering fundamental to advanced database concepts. The exercises are designed to build strong query-writing skills using real-world scenarios involving employees, departments, and transactional data.

The assignments focus on data retrieval, filtering, aggregation, joins, and advanced query techniques such as Common Table Expressions (CTEs).

---

## Assignments Included

### 1. SQL SELECT & WHERE

- Basic data retrieval using `SELECT`
- Filtering using `WHERE` conditions
- Use of operators: `AND`, `OR`, `NOT`
- Pattern matching using `LIKE`
- Range filtering using `BETWEEN`
- Handling distinct values using `DISTINCT`
- Column aliasing and expressions

These exercises involve working with employee, product, and order datasets to extract meaningful insights.

---

### 2. SQL JOIN & GROUP BY

- Inner joins across multiple tables
- Understanding relationships between entities
- Aggregation using `GROUP BY`
- Data summarization and reporting
- Working with multiple datasets (employees, departments, customers, orders)

These exercises demonstrate how to combine and analyze relational data effectively.

---

### 3. SQL CTE & Advanced Queries

- Use of Common Table Expressions (CTEs)
- Breaking down complex queries into readable steps
- Manager and department-based analysis
- Advanced filtering and ranking logic
- Real-world analytical queries

This section emphasizes writing clean and maintainable SQL for complex problems.

---

### 4. SQL Database Problem Solving

- Complex query-based problem solving
- Finding nth highest salaries
- Department-based filtering
- Date-based conditions
- Aggregation and reporting queries

This section focuses on solving real-world SQL interview-style questions using optimized queries.
