# Python Basics, Functions, Lists, and Dictionaries

## Overview
This project covers fundamental Python programming concepts, including syntax, variables, functions, lists, and dictionaries. It provides a strong foundation for understanding core programming principles and problem-solving using Python.

## Objective
- Understand Python syntax and variables
- Learn how to define and use functions
- Work with lists and perform operations on them
- Use dictionaries for structured data storage

## Technologies Used
- Python

## Files Included
- Shanaaz_Python_basics_syntax_variables.ipynb  
- Shanaaz_Python_Functions_Lists_Dictionaries.ipynb  

## Topics Covered
- Python syntax and basic structure
- Variables and data types
- Input and output operations
- Arithmetic operations
- Functions (definition, parameters, return values)
- Lists (creation, indexing, slicing, methods)
- Dictionaries (key-value pairs, access, update, iteration)

## How It Works
1. Learn basic Python syntax and variable usage
2. Implement functions for modular programming
3. Use lists to store and manipulate data
4. Use dictionaries to handle structured information
5. Combine concepts to solve basic programming problems
